# はじめに

Waterへようこそ。

Waterは、プログラミングを「作業」ではなく「呼吸」へと変えるために設計された、新しい概念のプログラミング言語です。

複雑な構文の壁を取り払い、ロジックが水面に広がる波紋のように自然に伝播していく体験を目指しています。開発者がコードを書くとき、その思考は透明であり、淀みなく流れるべきです。

## 核心となるコンセプト

Waterは以下の3つの哲学に基づいて構築されています。

### 1. 透明なロジック
コードは書かれることよりも読まれること、そして「理解されること」を優先します。Waterの構文は、自然言語に近い美しさと、数学的な厳密さを両立させています。

### 2. 波紋の伝播（リアクティビティ）
ひとつの{{評価|式を計算して値を導き出すこと|#}}は、静かな水面に落とされた一滴の雫です。その影響は、明示的な命令を待つことなく、依存するすべての要素へと自然に伝わっていきます。

### 3. 記述の呼吸
無駄な記号を削ぎ落とし、タイピングの心地よさを追求しました。Waterでコードを書くことは、深く息を吸い込み、思考を吐き出す行為そのものです。

:::info
Waterは現在プレビュー版です。仕様はコミュニティのフィードバックと共に、より澄んだものへと進化し続けています。
:::

## 次のステップ

Waterの世界をより深く知るために、以下のドキュメントから読み進めることをお勧めします。

* **[インストールガイド](#install)**: あなたの環境にWaterを導入します。
* **[変数と演算](#basics/variables)**: 基本的なデータの扱い方を学びます。
* **[チュートリアル](#notreadyet)**: 最初の「波紋」を作るためのステップバイステップガイドです。

:::warning
既存の命令型言語（C言語やJavaなど）の常識が、Waterでは通用しない場合があります。まずは頭を空にして、水の流れに身を任せてみてください。
:::
